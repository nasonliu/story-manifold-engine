## 2. 提取流程

### 2.1 骨架提取

```
小说文本 → 分章 → 章节摘要 → 关键情节点 → 结构化骨架
```

**提取字段**:
- `title`: 标题
- `author`: 作者
- `genre`: 类型
- `chapters`: 章节列表
  - `name`: 章节名
  - `summary`: 章节摘要 (200字)
- `structure`: 结构信息
  - `total_chapters`: 总章节数
  - `beats`:节点 (提取 6-10 个)
  - `tension_curve`: 情 张力曲线

### 2.2 提取方法

**方案 A: LLM 提取** (推荐)
- 使用 GPT-4 / Claude 提取结构
- 成本: ~$0.01/条
- 预计成本: $45

**方案 B: 规则 + 小模型**
- 规则提取章节转折点
- 小模型生成摘要
- 成本低但质量可能不足

## 3. 聚类分析

### 3.1 向量化

| 方案 | 模型 | 维度 | 状态 |
|------|------|------|------|
| 英文 | nomic-embed-text | 768 | ✅ 已有 |
| 中英混合 | bge-base-zh-v1.5 | 768 | ✅ 已有 |

### 3.2 降维与聚类

```python
# Pipeline
1. UMAP(n_components=50, n_neighbors=15, min_dist=0.1)
2. HDBSCAN(min_cluster_size=20, min_samples=5)
3. 可视化: UMAP → 2D/3D
```

### 3.3 分析维度

| 维度 | 指标 |
|------|------|
| 类型分布 | Genre/Archetype 聚类 |
| 文化差异 | 中/英/其他 聚类 |
| 结构模式 | beats 数量, tension curve 形状 |
| 长度分布 | 短篇/中篇/长篇 聚类 |
